Secure

Your College Board account information is only used temporarily to access your scores, and is not stored by EarlyScores.com.

Learn more about how Early Scores secures your data.

At Early Scores, security is our number one concern. We want to ensure that only you have access to only your scores. To make sure this is true, we utilize many industry standard hashing and encryption algorithms. Your username and password are never stored on our servers. Your scores are encrypted, and can only be viewed by you. If you would like to read more about the security in place with EarlyScores, please see our GitHub repository.

New: Temporary Password

Learn more about this advanced security feature

Early Scores is proud to announce a new feature for 2017 score fetching. If you are concerned about the safety of your account, Early Scores now supports temporary fetch mode. This simple mode will allow you to fetch scores using your temporary username and password even after you've reset your password on the College Board website. Follow these steps to enable this feature:

  1. Navigate to the College Board website. Sign in to your account.
  2. If did not follow the link above, navigate to the account settings page.
  3. Scroll down the page to change your password.
  4. Create a temporary password. Write down your temporary username and password for later use.
  5. Enable advanced settings and Temporary Password mode by clicking this link Sign in to Early Scores with your temporary password.
  6. Wait for your scores to be fetched and appear on the page.
  7. Head back to the College Board website and restore your real password.
  8. From now on, use your temporary username and password to log into Early Scores. Make sure the Fetch scores with temporary password. checkbox is checked, or your scores may be overwritten.

If you need to toggle the checkbox, see the "Advanced Settings" link on the right hand side.
If you would like to read more about the security in place with EarlyScores, please see our GitHub repository.
